# Interface JFrame Java

Projeto desenvolvido em Java utilizando JFrame no NetBeans, com foco na criação de uma interface gráfica para cadastro de usuários e gerenciamento de informações pessoais e de contato.

## 📌 Sobre o Projeto

Esta aplicação possui uma interface desktop criada com Java Swing/JFrame, permitindo o preenchimento de dados como:

- Código do usuário
- Nome
- Data de nascimento
- CPF
- RG
- Telefone
- Celular
- E-mail

O sistema também conta com botões separados para envio de:

- Dados pessoais
- Dados de contato

---

## 🖼️ Interface do Sistema

A interface foi construída utilizando componentes gráficos do Java Swing, organizados em seções para facilitar a experiência do usuário.

### Funcionalidades da Interface

- Cadastro de dados pessoais
- Cadastro de dados de contato
- Organização visual em painéis
- Campos formatados
- Botões de ação
- Ícones ilustrativos

## ▶️ Como Executar

### Pré-requisitos

- Java JDK instalado
- NetBeans IDE (recomendado)

### Executando

1. Clone o repositório:

```bash
git clone https://github.com/GuilhermeDevSoftware/Interface_Jframe_Java.git
```

2. Abra o projeto no NetBeans

3. Execute o projeto pressionando:

```text
Shift + F6
```

ou clique em:

```text
Run Project
```
